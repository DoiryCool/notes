### face_recognition
[face_recognition](https://github.com/ageitgey/face_recognition.git) 项目是世界上最简洁的人脸识别项目,可以仅凭一张人脸图片推理出其他图片。其深度模型为基于C++的dlib库中的深度学习模型。
